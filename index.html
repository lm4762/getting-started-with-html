What is a Web Page?

A web page is a text document written in HTML (HyperText Markup Language). Browers read the documents and interpret them so that they can be displayed.

The types of files found on the web are plain text, HTML, videos and music.

Tags and Elements

HTML documents consist of HTML elements. You can tell the browser what kind of element by using HTML tags. We learned about the following tags: bold, emphasis, anchor image, break, paragraph, span and div. Inline tages are a, span, br, img and strong. Block tags are div, p and form.

Computers are Stupid

Computers only understand certain language and syntax. If you don't write exactly what the computer understands it won't be able to figure out what you want to do.
